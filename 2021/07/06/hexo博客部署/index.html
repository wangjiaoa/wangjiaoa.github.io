hexo博客部署

hexo博客部署步骤—推到github端

  • 先在本地创建一个文件夹（路径随意），进入此文件夹下，打开cmd命令行模式，以下的所有操作均在此命令行下———->https://www.cnblogs.com/liuxianan/p/build-blog-website-by-hexo-github.html参考本篇博客

  • 安装Nodejs

    • node -v #查看node版本
    • npm -v #查看npm版本

  • 安装淘宝的cnpm 管理器

    • npm install -g cnpm –registry=http://registry.npm.taobao.org
    • cnpm -v #查看cnpm版本

  • 安装hexo框架

    • cnpm install -g hexo-cli
    • hexo -v #查看hexo版本
    • hexo init #生成博客 初始化博客

  • hexo操作

    • hexo s #启动本地博客服务
    • localhost:4000 #本地访问地址
    • hexo n 我的第一篇文章 #创建新的文章
    • hexo clean #清理
    • hexo g #生成

  • 本地博客推到github上

    • Github创建一个新的仓库 wangjiaoa.github.io

    • cnpm install –save hexo-deployer-git #在新建的目录下安装git部署插件

    • 配置_config.yml

    • hexo d #部署到Github仓库里

    • 浏览器访问wangjiaoa.github.io #访问这个地址可以查看博客

  • 更换主题

    • git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia #下载yilia主题到本地
    • #修改hexo根目录下的 _config.yml 文件 ： theme: yilia
    • hexo c #清理一下
    • hexo g #生成
    • hexo d #部署到远程Github仓库
    • 浏览器访问wangjiaoa.github.io #访问这个地址可以查看博客
