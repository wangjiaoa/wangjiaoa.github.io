office安装激活

office tool plus安装

  • 参考网站https://www.coolhub.top/archives/122

  • 安装步骤

    • 1、https://otp.landian.vip/zh-cn/官网下载office tool plus

    • 2、下载完整版，offcie-tool-v8.2,下载完之后直接解压，运行即可（建议管理员运行）——-如果双击office tool plus.exe无法运行，请尝试运行RunMe.bat

    • 3、安装之前记得卸载旧版本的office,并卸载wps；清除旧版本激活信息，Win10上必须做—-激活页面–>许可证管理—->清除激活状态）

    • 4、office部署安装—-解决弹窗问题—-使用批量版产品—-并且安装在企业长期版通道上。

激活步骤

  • 打开office tool(E:\awangjiao\OFFICE\Office Tool)—双击office tool plus.exe—清除激活状态（安装许可证）—KMS管理（复制宋发的链接–保存设置）—激活
